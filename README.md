
# Decibel

A music player app where you can have a playlist with many songs.


## Run Locally

Clone the project

```bash
  git clone https://github.com/alxmcr/fe-decibel-app-react-ts
```

Go to the project directory

```bash
  cd fe-decibel-app-react-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
